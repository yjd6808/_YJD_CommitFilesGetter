1. CommitFilesGetter 폴더를 git 경로에 둔 후 실행.bat 파일 실행

------------------------------------------------------------------------------------------------------

[Setting.ini파일]
저장 타입을 지정해주시면됩니다. 기본 값 : eAbsolutePath

[참고]
압축 푼 폴더를 로컬 깃 경로에 두시면 됩니다.

[사용처]
1. 여러 브랜치 작업시 브랜치간 이동(git checkout)이 제한될 때 ( 파일을 수정할 경우 ) 원격 저장소에 업로드된 데이터를 받아와서
테스트 하는 것이 가능합니다.
2. 과거의 리소스들을 checkout 없이 편하게 가져올 수 있습니다. ( 딱히 들고올일은 없을듯 )
3. 작업한 코드를 커밋하지 않고 소장하기 위함 ( GM아이템 메이커 소스를 작성했지만 반영안됨 )
